Paragraphs

What is selector’s specificity and who it’s calculated?

It’s a system used by CSS to determine which style rules apply when there is a conflict. It assigns scores to different types of selectors: IDs get the highest score, followed by classes and pseudo-classes, and then elements and pseudo-elements.

What are complex selectors?

These are selectors that combine two or more simple selectors to target elements more precisely. They use combinators like descendant, child, and sibling to specify the relationship between elements.

What is the difference between pseudo-classes and pseudo-elements?

Pseudo-classes are used to target elements in a specific state, such as :hover for when an element is hovered over. Pseudo-elements, like ::before, are used to style specific parts of an element, such as content before the element itself.

Blockquotes

A block quotation (also known as a long quotation or extract) is a quotation in a written document, that is set off from the main text as a paragraph, or block of text.

It is typically distinguished visually using indentation and a different typeface or smaller size quotation. It may or may not include a citation, usually placed at the bottom.
